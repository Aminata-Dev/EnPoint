# EnPoint

Get your English on Point with EnPoint : Auto-coaching to become fluent in english.

I am developing my own open-source app as a side project to improve my English vocabulary learning process. I want to bring together all the information I need (definitions, examples, but also elements more specific to my learning such as phonetic transcriptions of British pronunciation and videos of native speakers using the word/phrasal verb) in an easy-to-read and visually appealing dashboard (web application), as well as adding simple modules such as genrative AI-powered exercises or pronunciation practice with a speech-to-text. Ultimately, I’d like to be able to automatically save the words I’ve learnt so I can easily revisit them, freeing myself entirely from the app I currently use to manually record words I’ve come across but don’t yet know.
